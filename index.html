Important Information

This server officially supports Bedrock edition (console, mobile, etc.) but it is recommended you use Java edition when playing on the server as using Bedrock can be unstable at times. Java edition versions below 1.17.x won't support additions from the Caves and Cliffs update, therefore blocks below Y=0 won't be shown.

KeepInventory is OFF so if you die you will lose your stuff (unless you pick it up!). 

The server also has a whitelist. Any players who join on Day 1 will automatically be added to the whitelist. If you try to join and encounter a whitelist error, please contact me on Discord with your Minecraft username and I'll whitelist you. (ftt#8857)
